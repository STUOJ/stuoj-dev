| 英文              | 中文       | 解释                   |
|-----------------|---------|----------------------|
| judgement       | 评测点结果    | 单个评测点代码运行状态和评测结果    |
| submission      | 提交信息   | 题目提交信息、代码和所有评测点结果的汇总      |
| record      | 提交记录   | 某次提交代码的记录，record = submission + judgements      |
| language        | 语言     | 编程语言信息         |
| problem         | 题目     | 算法题目信息 |
| history | 题目历史记录 | 题目修改历史         |
| solution        | 题解     | 题目标准答案         |
| tag             | 标签     | 题目分类标签         |
| testcase        | 测试用例  | 代码评测使用的输入和预期输出数据  |
| user            | 用户     | 用户信息数据           |
